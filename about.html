📝 About This Application

This Progressive Web App (PWA) provides powerful audio transcription and AI-powered text processing capabilities right in your browser. Convert speech to text, then enhance the results with AI-powered proofreading, summarization, and more.

Designed for professionals, students, journalists, and anyone who needs to convert spoken content into polished written documents efficiently.

Version 1.0.0

✨ Key Features

  • 🎤
    Audio Transcription - Convert speech to text using OpenAI's Whisper technology
  • 🔍
    Proofreading - Fix grammar, spelling, and punctuation errors automatically
  • 📋
    Meeting Minutes - Transform raw transcripts into professional meeting minutes
  • 📝
    Summarization - Extract key points from long transcripts
  • 📑
    Outline Generation - Create structured outlines from unstructured content
  • 📱
    PWA Support - Installable on your device for offline use

🛠️ Technical Details

This application leverages cutting-edge AI technologies to deliver accurate results:

OpenAI Whisper
GPT-3.5/4
Progressive Web App
Web Audio API
Service Workers
LocalStorage

All processing happens securely via OpenAI's API. Your data is never stored on our servers.

🔒 Privacy Policy

Data Handling

Your audio files and transcripts are processed directly in your browser or sent to OpenAI's API for processing. No data is stored on any intermediary servers.

API Keys

Your OpenAI API key is stored locally in your browser's storage and is only used to make direct requests to OpenAI's API. We never see or store your API key.

Cookies

This application uses localStorage to remember your settings but does not use tracking cookies.

Offline Use

When installed as a PWA, the application can work offline for previously accessed content, but transcription and AI features require an internet connection.
